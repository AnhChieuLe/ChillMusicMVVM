# Chill Music

Ứng dụng nghe nhạc offline

Dynamic theme - theme thay đổi theo từng bài hát

MVVM + LiveData + Databinding

Rebuild from [Chill Music MVC](https://github.com/AnhChieuLe/ChillMusic)
